# Ramadan Kids Corner

A class project built with AI assistance to practice and develop our prompting skills. The goal was to use AI tools throughout the development process — from design decisions to code generation — and learn how to write effective prompts to get useful results.

The project itself is an educational website for kids (6–12 years old) to discover Ramadan in a fun and interactive way.

---

## Pages

- **Home** — hero section, activity cards, call-to-action
- **Activities** — coloring, recipes, crafts with Bootstrap modals
- **Stories** — 5 complete stories with morals, accordion layout
- **Quiz** — interactive quiz system with JSON data, scoring, and progress tracking
- **Contact** — form with real-time validation, Google Maps, FAQ

## Run locally

```bash
git clone https://github.com/bahasalah255/islam-kids
cd islam-kids
python -m http.server 8000
```

Then open `http://localhost:8000` — no dependencies to install, everything loads via CDN.

---

## About the AI part

This was a school assignment where we had to use AI to build a real project. We worked on writing clear, structured prompts to generate code, fix bugs, and improve the UI — and iterated based on what the AI gave us. Good practice for working with AI tools in a real dev workflow.
